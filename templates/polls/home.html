🏆 Polls App API

Welcome to the Polls App API. Use the following endpoints:

📋 Polls

GET /api/polls/ - List all polls

POST /api/polls/ - Create a new poll

GET /api/polls/1/ - Get poll details

GET /api/polls/1/results/ - Get poll results

🗳️ Voting

POST /api/polls/1/vote/1/ - Vote on a poll

👤 User

GET /api/my-polls/ - Get user's polls

🔧 Admin

/admin/ - Django Admin

Quick Start:

  • Visit Admin to manage polls
  • Use /api/polls/ to interact with the API
  • Use tools like Postman or curl to test API endpoints
